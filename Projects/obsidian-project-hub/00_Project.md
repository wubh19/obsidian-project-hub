# obsidian-project-hub

## 简介

当前插件仓库本身作为第一个受管项目，用于验证项目面板、版本中心和任务看板。

## 当前目标

- 统一三状态（Todo/In Progress/Done），移除 Blocked
- 项目/版本/任务支持工时统计
- 任务工时解析与聚合展示
- 版本工时支持 frontmatter
- 默认项目数据与文档保持同步

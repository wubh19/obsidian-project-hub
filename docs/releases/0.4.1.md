### Added

- 新增当前仓库项目数据，可直接作为第一个 obsidian-project-hub 项目使用。
- 新增自动部署脚本，发布完成后可一键同步插件、项目和 demo 到本地 Vault。

### Changed

- 项目解析支持第二层级目录推断，不再只依赖 Projects/项目名 结构。
- Dashboard 页面改为更紧凑的布局。

### Fixed

- 修正快速新建任务写入路径，改为跟随项目真实目录。

### Notes

- 发布前执行 npm.cmd run release，脚本会依次构建、检查版本与文档、再同步到 Obsidian Vault。
- 本地部署前先复制 obsidian-project-hub.deploy.example.json 为 obsidian-project-hub.deploy.local.json 并填写 Vault 路径。
